Dashboard Hotel

Dashboard Hotel is an administration platform designed specifically for the managers of Hotel Miranda. This advanced tool allows adding, editing, and deleting employees, rooms, reservations, and managing customer messages coming from the Hotel Miranda project. Administrators can centrally manage reservations, processing customer requests in real-time as messages, directly linked to the hotel’s booking platform. The main dashboard offers a clear and functional view of the reservation calendar, where entry and exit dates, daily occupancy, and room statuses can be easily managed. This streamlines decision-making and ensures precise tracking of the hotel’s daily operations.

View Proyect

Built with modern technologies like React, Styled Components, TypeScript, Redux, Express, Node.js, and MongoDB, Dashboard Hotel provides a robust, dynamic, and secure environment. API creation and consumption ensure the platform is flexible and scalable, offering a fast and tailored experience to meet the hotel's needs.
